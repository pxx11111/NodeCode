CodePad About

Welcome to CodePad About.
CodePad is a simple source code viewer that supports syntax highlighting. it supports all C-like, Bash-like, and XML-like languages. it automatically selects the syntax depending on the file.

CodePad uses WebKit and google-code-prettify to do the syntax highlighting.
